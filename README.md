# AI Plant Care Tool

Senior Design project to help users care for common houseplants using image analysis + user-reported symptoms.

## Team

- Brady Hajec

## Current Scope (MVP)
- Upload plant photo + optional symptom text
- Identify plant species (initially: common houseplants)
- Detect common stress/symptom indicators
- Provide basic care recommendations
- Reminders and feedback loop (stretch)
